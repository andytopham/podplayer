README for podplayer.
Starting with Raspbian.
Connect ethernet cable and plug in wifi adapter and DAC.
Boot.
Scan for new raspberry pi ip.
Putty into this ip.
Winscp into this ip.
mkdir ~/podplayer
setup winscp to keep remote dir up to date.
chmod +x installer.sh
./installer.sh (takes a long time to run!)
Make updates described at the end of the installer.
Reboot.
